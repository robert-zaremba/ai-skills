# Sui Move

Sui Move is a resource-oriented smart contracts programming language for Sui Blockchain.

## Features

- **Complete Language Reference**: Types, abilities, modules, functions, control flow
- **Object Model**: Ownership types, storage functions, transfer restrictions
- **Design Patterns**: Capability, Witness, OTW, Hot Potato, Publisher patterns
- **Anti-Patterns**: Common mistakes and how to avoid them
- **Standard Library & Sui Framework**: Quick reference for all modules
- **Testing Guide**: Unit tests, test scenarios, coverage, gas profiling
- **Best Practices**: Code quality checklist, naming conventions
- **CLI Reference**: All essential commands for development

## File Size

The skill is approximately 2,800 lines - comprehensive enough to cover all aspects of Sui Move development while remaining focused and actionable.
